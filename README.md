## Usage

### install package

```bash
npm i -D eslint @mutoe/eslint-config-preset
# or
yarn add -D eslint @mutoe/eslint-config-preset
```

### use in config js

```javascript .eslintrc.js
module.exports = {
  "extends": "@mutoe/eslint-config-preset"
}
```
